Strongly Connected Component

コードについての説明

有向グラフの強連結成分分解を行うアルゴリズム. また強連結成分分解後の各強連結成分を頂点とする DAG を, 頂点をトポロジカル順序に番号付けした形で求める.
強連結成分分解は DFS 2 回で求めることができ, 1 回目の DFS では適当な頂点から始めて帰りがけ順に番号を振っていく. 2 回目の DFS ではもとのグラフの辺をすべて逆にしたグラフ上でまだたどっていない頂点で先ほど振った頂点番号が大きいものからたどっていく. このときたどることのできた頂点集合が 1 つの強連結成分に対応し, これを繰り返す.
2 回目の DFS ではトポロジカル順序に沿って強連結性成分を探しているので各強連結成分を頂点とする DAG の頂点の番号付けも同時に行える.

(関数)
add_edge(u, v): 有向辺(u,v) を追加する
make_graph(): solve() 後に呼び出すと各強連結成分を頂点とする DAG が graph に入る

時間計算量: O(n+m)

コード

class SCC {
public:
	int V,cnt;
	vector<vector<int> > G,rG,graph;
	vector<int> vs,cmp;
	vector<bool> used;
	SCC(int node_size) : V(node_size), G(V), rG(V), cmp(V), used(V, false){}
	void add_edge(int from,int to){
		G[from].push_back(to);
		rG[to].push_back(from);
	}
	void dfs(int u){
		used[u] = true;
		for(int v : G[u]){
			if(!used[v]){
				dfs(v);
			}
		}
		vs.push_back(u);
	}
	void dfs(int u,const int k){
		used[u] = true;
		cmp[u] = k;
		for(int v : rG[u]){
			if(!used[v]){
				dfs(v,k);
			}
		}
	}
	int solve(){ //強連結成分の数を返す
		for(int i = 0; i < V; i++){
			if(!used[i]){
				dfs(i);
			}
		}
		fill(used.begin(),used.end(),false);
		cnt = 0;
		for(int i=V-1;i>=0;i--){
			if(!used[vs[i]]){
				dfs(vs[i],cnt++);
			}
		}
		return cnt;
	}
	void make_graph(){
		graph.resize(cnt);
		for(int i = 0; i < V; i++){
			for(int v : G[i]){
				if(cmp[i] != cmp[v]){
					graph[cmp[i]].push_back(cmp[v]);
				}
			}
		}
	}
};
