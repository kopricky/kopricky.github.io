$C_3$ Find Algorithm

個人的メモ

threshold $α$ を決めて, 次数が $α$ 以下の各頂点についてはそこから出る異なる $2$ 辺についてもう一方の端点同士の間に辺があるかどうかを調べる(あれば true).
次数が $α$ より大きい頂点(個数は $2m / α$ 以下)については隣接行列の $3$ 乗($64$ bit高速化が使える)を計算して対角成分を見る(非ゼロの成分があれば true).
(補足) $C_3$ なら次数の小さい頂点からそれを含む $C_3$ の数を数えていくことで $\widetilde{\O} (m^{1.5})$ で finding だけでなく counting もできるのでそちらのほうが良いかもしれない. 上記論文はより一般に, 与えられた $k$ について $C_k$ の find と count を効率良く行うアルゴリズムを与えている.
