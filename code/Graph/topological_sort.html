Topological Sort

コードについての説明

有向グラフがDAG(閉路を持たないグラフ)であるかの判定およびDAGの場合は頂点をトポロジカル順序にソートするアルゴリズム.
既に探索し終わった頂点からしか辺が入ってこないような頂点から順に隣接する頂点を探索していく.

(関数)
solve(): DAGかどうか(true/false)を返し, true なら res にトポロジカル順序にソートされた頂点列が格納される.

時間計算量: O(m)

コード

class tsort {
public:
	int V;
	vector<vector<int> > G;
	vector<int> deg,res;
	tsort(int node_size) : V(node_size), G(V), deg(V, 0){}
	void add_edge(int from,int to){
		G[from].push_back(to);
		deg[to]++;
	}
	bool solve() {
		queue<int> que;
		for(int i = 0; i < V; i++){
			if(deg[i] == 0){
				que.push(i);
			}
		}
		while(!que.empty()){
			int p = que.front();
			que.pop();
			res.push_back(p);
			for(int v : G[p]){
				if(--deg[v] == 0){
					que.push(v);
				}
			}
		}
		return (*max_element(deg.begin(),deg.end()) != 0);
	}
};
