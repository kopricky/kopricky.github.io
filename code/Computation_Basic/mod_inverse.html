Mod Inverse

コードについての説明

Z/pZ (p は素数) の逆元を拡張ユークリッドの互除法を用いて計算するアルゴリズム.
フェルマーの小定理を用いて p-2 乗を計算して求めても良い.

(関数)
mod_inverse(a, m): a の Z/mZ 上の逆元を求める(m は素数とする)

時間計算量: O(log a)

コード

void extgcd(int a,int b, int& x,int& y)
{
	if(b != 0){
		extgcd(b, a % b, y, x);
		y -= (a / b) * x;
	}else{
		x = 1;
		y = 0;
	}
}

int mod_inverse(int a, int m)
{
	int x, y;
	extgcd(a, m, x, y);
	return (m + x % m) % m;
}
