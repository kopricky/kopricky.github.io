LIS 2D

コードについての説明

2次元LIS とは {(x_1, y_1), (x_2, y_2), ... , (x_n, y_n)} (x_i <= x_i+1 かつ y_i <= y_i+1 かつ (x_i, y_i) ≠ (x_i+1, y_i+1)) というような単調増加する組の列のことをいう(イコール周りは定義がいろいろ)
本実装では {(x_1, y_1), (x_2, y_2), ... , (x_n, y_n)} (x_i <= x_i+1 かつ y_i >= y_i+1 かつ (x_i, y_i) ≠ (x_i+1, y_i+1)) というような x の増加に伴い, y は単調に減少する列を動的に管理するデータ構造になっている
これは 2 次元座標上にプロットしたとき追加された点の下側 envelope を保持していることに等しい.

(関数)
insert(val): 点 val (∈ R^2) を追加する. 実際に下側 envelope に組み込まれたか(true / false)を返す.

時間計算量: insert ならしO(logn)

コード

template<typename T> class LIS2D
{
private:
    using ptt = pair<T,T>;
    set<ptt> st;
public:
    //実際に追加されたかどうかを返す
    bool insert(pair<T,T> val){
        bool in = false;
        auto it = st.lower_bound(val);
        if(it != st.end()){
            if(*it == val) return in;
            in = (it->second >= val.second);
        }
        if(in){
            while(it != st.end() && it->second >= val.second){
                it = st.erase(it);
            }
        }else{
            if(it != st.begin()){
                it--;
                in = (it->second > val.second);
            }else{
                in = true;
            }
        }
        if(in) st.insert(val);
        return in;
    }
    void print(){
        for(auto& val : st){
            cout << "{" << val.first << "," << val.second << "}" << " ";
        }
        cout << "\n";
    }
};
