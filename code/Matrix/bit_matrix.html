Bit Matrix

コードについての説明

F_2上の行列の積に (or, and) をのせた演算が 64 bit高速化により O(n^3) よりも速くなるという話.
具体的な使い道としては隣接行列の積が高速に計算できるようになるため到達可能性などの問題に有効.

時間計算量: O(n^3/64) (64 bit環境で)

コード

#define MAX_N 2000

class mat {
public:
    bitset<MAX_N>* a;
    mat(){ a = new bitset<MAX_N>[MAX_N]; }
    mat operator*(const mat& m) const {
        mat ans;
        for(int i = 0; i < MAX_N; i++){
            for(int j = 0; j < MAX_N; j++){
                if(this->a[i][j] == 0) continue;
                ans.a[i] |= m.a[j];
            }
        }
        return ans;
    }
};
