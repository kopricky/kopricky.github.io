BIT

コードについての説明

BIT(binary indexed tree) は1点加算と区間和のクエリを高速に処理することのできるデータ構造です. segment tree でも実装できますが BIT の方が基本的に速いです.
fenwick tree とも呼ばれています. 説明はおそらく検索すればわかりやすいものがいくつも出てくると思うので他のかたにゆずります...
ブロックの持ち方をフィボナッチ数にすることでlogの底が変わり計算量的には速くなるという話もあります(参考). (実測はたぶん速くならない)

時間計算量: 構築 O(n), 各クエリ O(logn)

コード

class BIT:
    def __init__(self, node_size):
        self._node = node_size+1
        self.bit = [0]*self._node

    def add(self, index, add_val):
        index += 1
        while index < self._node:
            self.bit[index] += add_val
            index += index & -index

    def sum(self, index):
        index += 1
        res = 0
        while index > 0:
            res += self.bit[index]
            index -= index & -index
        return res
