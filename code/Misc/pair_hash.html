Pair Hash

コードについての説明

std::pair のハッシュ値はデフォルトで定義されていないため, unordered_set, unordered_map などを使いたいときは以下の実装を用いると良い(boost の実装を基に書いた).
よくハッシュ値をそのまま xor で結合しているものを見るが, 自明に衝突するケース(同じ要素の組だった場合)が存在するのでやめたほうがいい.

コード

struct pair_hash {
    template <class T1, class T2>
    unsigned int operator() (const pair<T1, T2>& p) const {
        unsigned int lhs = hash<T1>()(p.first), rhs = hash<T2>()(p.second);
        return lhs^(rhs+0x9e3779b9+(lhs<<6)+(lhs>>2));
    }
};
